# Todo app

# Set up backend
```
$ python -m venv venv
$ venv\Scripts\activate
$ pip install -r requirements.txt
```

# Setup frontend
```
$ cd frontend
$ yarn
```
# Start project
## Frontend
```
$ cd frontend
$ yarn start
```
## Backend
```
$ cd backend
$ python main.py
```
## Follow web UI at http://localhost:3000/
## Follow web API at http://localhost:8000/docs
